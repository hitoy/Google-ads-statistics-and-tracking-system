谷歌竞价广告点击攻击跟踪系统
作者: Hito(http://www.hitoy.org/) 王荣杰
功能: 
跟踪Google竞价（联盟和搜索）点击，记录点击用户IP，语言，来源，访问页面等信息，并简单分析用户点击情况，对可疑广告点击作出标记。

使用方法:
1,用database.sql创建数据库结构
2,在config.phpz中设置数据库相关信息和竞价区分字段，登陆口令等等
3,在ba.js中，更改dataurl为搭建网站的ba.php的url,更改 click_key为竞价的区分字段，更改ads_site_key为提取联盟网站的正则表达式
4,系统搭建好之后，访问cat.php就可以查看相关信息

此工具也适用于统计其它竞价广告的点击情况
